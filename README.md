Zero Mining Dashboard

(Custom WebUI for CKPool)

Setup:

#Clone Zero Mining GitHub:

sudo git clone https://github.com/S7nAcks/zeromining.git

cd zeromining

#Install Dependencies:

sudo npm init -y

sudo npm install express

sudo npm install node-fetch

sudo npm install dotenv

#Run:

node server.js

#Open:

http://localhost:3000
